Это способность программы идентично использовать объекты с одинаковым интерфейсом без информации о конкретном типе этого объекта. Это определение параметрического полиморфизма, конкретная же реализация параметрического полиморфизма, являются, например [[Generic]] классы 

```csharp
interface IBook<T>
{
	T GetHeader();
}
```

В данном случае мы можем поставить вместо T любой тип, или вообще убрать T и применить обычные типы
